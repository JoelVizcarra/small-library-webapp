# Small-library-webapp
Small webapp that manage books in a library, also had user authentication via JWT.
Backend: NodeJS 
Frontend: React

Server side dependences:
  - Express
  - Express-validator
  - jsonwebtoken
  - mongoose
  - bcrypt-nodejs
  - morgan
  - multer
  - body-parser

Client side dependences:
  - react, react-dom, react-scripts
  - bootstrap@4.1.1
